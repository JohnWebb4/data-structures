# Data Structures

## Goal

Implementing different data structures in C++

## Setup

Install [GTest](https://github.com/google/googletest)

Install [CMake](https://cmake.org/)

Clone the project

```sh
git clone https://github.com/JohnWebb4/data-structures.git
cd data-structures
```

Run setup

```
./setup.sh

```

Build the project

```sh
cmake .
```
